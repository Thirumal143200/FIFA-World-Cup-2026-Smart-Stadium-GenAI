## [1.0.0] - 2026-07-15

### Added
- Animated role selection landing page (Fans, Staff, Organizers, Security).
- 8 Google Gemini-powered GenAI agents for navigation, crowd predictions, multilingual chat, SOS emergency triage, transit planning, and operational support.
- Fully simulated interactive maps, crowd heatmaps, and carbon footprint calculators.
- Real-time incident logs backing both Firebase Firestore and memory database fallbacks.
- Global dashboard layout with responsive header, sidebar, and mobile tab bar.
- Integrated high-contrast and light/dark theme providers.
- Strict security middleware including CSP headers, CORS whitelist, and rate limiting.
- Complete documentation suite (README, ARCHITECTURE, DEPLOYMENT, etc.).
